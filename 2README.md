# university Department system

## project title and purpose
This is a simple university department management system python project. it's core functionalities is to manage students, courses, and enrollment within a university department.
the system depicts a well OOP, file handling and user interactions.

## instructions for installation and excecution
1. Make sure you have python installed on your PC and up to date
2. download this repository
3. open the project folder in your terminal
4. run the interface file:

   python interface.py

## Example usage
   - Enter Student Name and ID
   - add a course name, course code and credits
   - choose between enrolling the student or not
   - Save student data to a csv file
   - load student data from teh csv file
  
## key features
   - add, manage and view students informations
   - add and manage courses
   - enrol students in courses
   - save and load students using csv files
   - simp,e user input interaction
